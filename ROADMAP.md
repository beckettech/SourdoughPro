# Sourdough App — Project Roadmap

## Phase 1: Foundation (Weeks 1-3)
- [ ] Tech stack decision & project scaffolding
- [ ] Core data models (Starter, FeedingLog, Recipe, User)
- [ ] Auth (email/password, Apple/Google OAuth)
- [ ] Basic UI shell & navigation
- [ ] Starter CRUD (create, name, edit, delete)

## Phase 2: Core Tracking (Weeks 4-6)
- [ ] Feeding cycle logging (time, flour type, hydration ratio)
- [ ] Feeding reminders/notifications (push + local)
- [ ] Starter health dashboard (activity graph, streak tracking)
- [ ] Starter discard calculator

## Phase 3: AI Vision MVP (Weeks 7-9)
- [ ] Photo capture & upload flow
- [ ] Starter health scoring (bubble activity, rise height, color)
- [ ] Loaf scoring (crumb, crust, shape, oven spring)
- [ ] Basic AI feedback text generation

## Phase 4: Recipes & Content (Weeks 10-11)
- [ ] Built-in recipe library (20+ curated recipes)
- [ ] Custom recipe creation
- [ ] Baker's percentage calculator
- [ ] Timers & step-by-step baking guides

## Phase 5: Pro Features (Weeks 12-14)
- [ ] AI coaching chat (context-aware baking advice)
- [ ] Custom recipe generation from ingredients on hand
- [ ] Smart notifications (weather-aware, schedule-aware)
- [ ] Unlimited starters (gated by tier)

## Phase 6: Launch (Weeks 15-16)
- [ ] App Store & Play Store submission
- [ ] Landing page & waitlist
- [ ] Social media content strategy (TikTok/IG)
- [ ] Beta testing with sourdough communities

## Key Milestones
| Milestone | Target |
|---|---|
| Prototype (starter tracking) | Week 3 |
| Internal alpha | Week 6 |
| AI vision working | Week 9 |
| Feature-complete beta | Week 13 |
| Public launch | Week 16 |

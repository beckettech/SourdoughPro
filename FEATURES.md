# Sourdough App — MVP vs Nice-to-Have Features

## MVP (Ship in 16 weeks)

### Must-Have
- [ ] User auth (email, Apple, Google)
- [ ] Create & name starters
- [ ] Log feedings (timestamp, flour type, water temp, hydration %)
- [ ] Feeding reminders (push notifications, configurable schedule)
- [ ] Starter health timeline (visual log of feedings, rise observations)
- [ ] AI vision: starter health score from photo
- [ ] AI vision: loaf score from photo (crumb, crust, shape)
- [ ] 10-20 curated recipes with baker's percentages
- [ ] Basic baker's percentage calculator
- [ ] Subscription paywall (Free / Pro / Baker tiers)
- [ ] Onboarding flow

## Nice-to-Have (Post-Launch)

### AI Coaching
- [ ] Conversational AI baking coach (chat interface)
- [ ] "What can I bake?" from ingredients on hand
- [ ] Custom recipe generation
- [ ] Troubleshooting wizard ("my starter won't rise")

### Social & Community
- [ ] Share bakes to feed / community gallery
- [ ] Follow other bakers
- [ ] Weekly bake challenges
- [ ] Leaderboards / streaks

### Advanced Tracking
- [ ] Temperature & humidity logging (auto from weather API)
- [ ] Weight-based tracking (scale integration via Bluetooth)
- [ ] Predictive feeding schedule (ML model)
- [ ] Sourdough discard recipe suggestions
- [ ] Calendar view of baking schedule

### Content & Monetization
- [ ] Video tutorials (embedded)
- [ ] In-app purchases (equipment, ingredients via affiliates)
- [ ] Pro community / forums
- [ ] Branded merch store

### Platform
- [ ] Web app (PWA or full web)
- [ ] Apple Watch complications (feeding reminders)
- [ ] Shortcuts / Siri integration
- [ ] Widget (home screen starter status)

## Cut for Now
- Social features (high scope, low revenue impact at launch)
- Hardware integrations (scale, thermometer)
- Web app (mobile-first)
- Video content (licensing complexity)

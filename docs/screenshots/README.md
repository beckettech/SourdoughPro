## How to capture

1. Build & run on iPhone 16 Pro simulator (preferred — matches App Store dimensions)
2. `⌘S` in the simulator saves a screenshot to your Desktop
3. Drop the files into this folder and push

## App Store sizes

For App Store Connect you'll also need:
- `6.9inch` (iPhone 16 Pro Max) — 1320×2868
- `6.5inch` (iPhone 15 Plus) — 1284×2778
- `5.5inch` (iPhone 8 Plus) — 1242×2208

# Sourdough App

AI-powered sourdough starter & baking assistant.

## Overview

Helps bakers track fermentation, manage starters, and troubleshoot their bakes with AI-powered insights.

## Features (Planned)

- Starter maturity tracking
- Fermentation progress AI (photo analysis)
- Temperature-adjusted timing
- Visual bulk fermentation guide
- "What went wrong?" troubleshooting
